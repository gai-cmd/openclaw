# Trylot 서비스 와이어프레임 스케치

## 홈 페이지
- **헤더**: 로고, 내비게이션 메뉴 (홈, 서비스, 가격, 문의)
- **메인 배너**: 서비스 소개 이미지, 주요 메시지 버튼
- **서비스 섹션**: 서비스 특징 카드 3개, 각 카드 이미지 및 설명 포함
- **리뷰 섹션**: 사용자 후기 슬라이더 (3개 샘플)
- **푸터**: 연락처 정보, 소셜 미디어 링크

## 로그인 페이지
- **헤더**: 로고
- **로그인 폼**: 이메일 입력 필드, 비밀번호 입력 필드, 로그인 버튼
- **비밀번호 찾기** 링크
- **회원가입 계정 만들기** 링크

## 대시보드
- **사이드바**: 내비게이션 메뉴 (대시보드, 프로파일, 설정)
- **메인 화면**: 통계 그래프, 최근 활동 로그 리스트
- **요약 카드**: 계정 정보, 액티브 상태

## 서비스 페이지
- **헤더**: 로고, 내비게이션 메뉴
- **서비스 목록**: 각 서비스 카드, 이미지 및 설명 포함
- **필터 및 정렬 옵션**

## 문의 페이지
- **헤더**: 로고, 내비게이션 메뉴
- **문의 폼**: 이름, 이메일, 메시지 입력 필드, 제출 버튼
- **FAQ 섹션**: 자주 묻는 질문 리스트 및 답변

## 디자인 요소
- **컬럼 그리드**: 12 컬럼 레이아웃
- **글꼴**: 'Roboto', sans-serif
- **버튼**: 활성화 상태 (#6200EE, #FFFFFF), 비활성화 상태 (#E0E0E0, #9E9E9E)
