# 칸반차트 UI/UX 개선 및 기초 디자인 문서

## 1. UI 분석
- **배경 색상**: 여러 상태를 구분하는 데 명확한 대비 제공, Primary Color(#6200EE)와 Background Color(#F5F5F5)의 대비 활용
- **타이포그래피**: 각 컬럼과 카드에서 명확한 정보 전달을 위해 기본 폰트 적용
- **버튼 스타일**: 카드 추가 및 이동 버튼에 명확한 활성화 색상과 크기 제공

## 2. UX 개선 방향
- **드래그 앤 드롭**: 사용자 경험을 최적화하기 위해 부드러운 트랜지션(0.3s)을 적용
- **정보 가독성**: 주요 작업 정보는 굵은 텍스트로 강조, 부가 정보는 캡션 스타일 사용

## 3. 접근성 고려사항
- **대비 비율 유지**: 최소 4.5:1의 대비 비율을 활용하여 시각적 접근성 개선
- **폰트 크기 조정**: 최소 12px를 유지하여 정보의 가독성을 높임

## 4. 레이아웃 및 기타 디자인 요소
- **그리드 시스템**: 12 Column Grid 사용하여 일관된 레이아웃 제공
- **공간 및 정렬**: 기본 16px 패딩과 마진을 적용하여 콘텐츠 간의 적절한 간격 유지

## 5. 차트 구성
- **카드 요소**: 각각의 할일 카드에는 제목, 설명, 담당자 정보가 포함됨
- **컬럼 상태**: 할일, 진행 중, 완료로 명확하게 분류하여 사용자가 현재 진행 상태를 쉽게 파악할 수 있도록 함

## 결론

이 문서는 칸반차트의 UI/UX 개선을 위해 기존 Trylot 디자인 시스템과 와이어프레임 스케치를 참고하여 작성되었습니다. 지속적인 피드백을 통해 개선점을 찾아나갈 예정입니다.
