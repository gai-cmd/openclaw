# Kanban Chart Design System

## Colors
- **Priority Badges**
  - Critical: #FF0000
  - High: #FF7F00
  - Medium: #FFFF00
  - Low: #00FF00

- **Status Columns**
  - Backlog: #D3D3D3
  - To Do: #ADD8E6
  - In Progress: #87CEFA
  - Review: #EE82EE
  - Done: #98FB98
  - Blocked: #FFA07A

- **Phase Indicators**
  - P0: #1E90FF
  - P1: #00CED1
  - P2: #32CD32
  - P3: #FFD700
  - P4: #FF4500

- **Domain Tags**
  - DOC: #FF6347
  - UI: #4682B4
  - FE: #7B68EE
  - BE: #9ACD32
  - DB: #6B8E23
  - QA: #FF69B4
  - OPS: #B0E0E6
  - MKT: #DDA0DD

## Typography
- **Font Family:** Arial, sans-serif
- **Font Sizes:**
  - Header: 24px
  - Subheader: 18px
  - Body: 14px

## Spacing
- **Margins:** 16px
- **Padding:** 16px

## Components
- **Card Design**: Each task card includes task_id, title, assignee, due_date, priority badge, and a progress bar. Maintain a consistent style across all cards.
- **Filters**: Include project selector, phase filter, domain filter, and assignee filter for enhanced user accessibility.

This design system ensures accessibility and a consistent user experience across the Kanban chart project.
