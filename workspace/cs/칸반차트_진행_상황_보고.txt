칸반차트 프로젝트는 현재 Phase 1이 진행 중이며, 주요 작업 항목은 다음과 같습니다:

1. DB 스키마 및 설계서 기반 데이터베이스 생성
2. Task CRUD API 구현 작업 (Hono REST)
3. 텔레그램 봇 명령어 구현 (`/task create/update/list/view`, `/board`)
4. Task ID 자동 생성 로직 개발
5. 기본 칸반 보드 텍스트 뷰 구현 (텔레그램 내)

현재 진행률은 약 50%로, 주요 기능들이 개발 진행 중에 있습니다.

다음 단계는 Phase 2: 웹 대시보드의 개발로, React 기반 칸반 보드 UI 구현과 관련된 작업을 포함하며, 드래그 앤 드롭 Task 이동, 필터링/검색 기능, 프로젝트 대시보드 위젯 구현 등이 예정되어 있습니다.
